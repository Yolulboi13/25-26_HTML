Tables

The table opens a table. Each row is used with tr tag, and columns use td.

Tables have attributes, like:

  • border - sets a border for the table
  • width - sets a width for the table or row
  • height - sets a height for the table or column
  • th - set headers at the top
  • colspan - sets number of columns to merge
  • rowspan - sets number of rows to merge
